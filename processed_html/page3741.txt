MXML File Extension - What is an .mxml file and how do I open it? Component used by a Adobe Flex application; written in the MXML markup language, which is similar to  XML , but was developed by Macromedia; used for laying out user interface components in the application. Most Flex applications include multiple MXML and ActionScript ( .AS ) files. Our goal is to help you understand what a file with a *.mxml suffix is and how to open it. The Flex MXML Component file type, file format description, and Mac and Windows programs listed on this page have been individually  researched and verified by the FileInfo team .  We strive for 100% accuracy and only publish information about file formats that we have tested and validated.
