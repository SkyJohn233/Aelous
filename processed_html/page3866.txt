Tables   Python-Markdown 3.1.1 documentation The Tables extension adds the ability to create tables in Markdown documents. This extension is included in the standard Markdown library. Tables are defined using the syntax established in  PHP Markdown Extra . Thus, the following text (taken from the above referenced PHP documentation): will be rendered as: See  Extensions  for general extension usage. Use  tables  as thename of the extension.
